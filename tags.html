Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block elements

Tag Name Purpose
h1–h6 headings include a heading
p paragraph include a paragraph of text
table table include a table
th table heading include a table heading
tr table row include a table row
td table data include a table data cell
tfoot table footer include a table footer
div division create a division, easy to get styled with css

Inline elements

Tag Name Purpose Example Result
em emphasize emphasize a portion of text <em>technical sophistication</em> technical sophistication
u underline underline a portion of text <u>technical sophistication</u> technical sophistication
span separate a portion of text div equivalent, inline version <span>technical sophistication<span> technical sophistication
